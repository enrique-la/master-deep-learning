# Deep Learning — Máster en IA & Data Science

Asignatura de redes neuronales profundas con proyectos de visión por computador y renderizado neuronal 3D.

## Contenido

### Fundamentos de Redes Neuronales
- Feedforward Neural Networks (FNN)
- Backpropagation y optimización
- Overfitting, regularización (Dropout, Batch Norm)
- MNIST digit recognition

### Visión por Computador con OpenCV (10 Labs)
- Operaciones básicas y matemáticas sobre imágenes
- Thresholding, filtrado, detección de bordes
- Erosión, dilatación, opening/closing

### Redes Convolucionales (CNN)
- Clasificación de cartas con CNN
- Visualización de filtros aprendidos
- Transfer Learning (MobileNet, ViT)

### Detección de Objetos
- Single object detection
- Multiple object detection en placas PCB
- Segmentación de imágenes

### Redes Recurrentes (RNN)
- Procesamiento de secuencias temporales

### NeRF & Gaussian Splatting
- Neural Radiance Fields (NeRF) para renderizado 3D
- Gaussian Splatting con nerfstudio
